### Required

> Ubuntu

```
sudo apt-get update && sudo apt-get install -y vim
```

> Big Sur

```
brew install vim
```

### Install

```
bash <(curl -fsSL https://raw.githubusercontent.com/liuchengxu/space-vim/master/install.sh)
```

**layers**

```
vim ~/.spacevim
```

```
'html', 'ruby', 'javascript', 'file-manager', 'code-snippets',
```

> Reload(需要先保存)

`Space` -> `f` -> `R`

### vim-plug

* 状态查看: `PlugStatus`
* 安装插件: `PlugInstall`
