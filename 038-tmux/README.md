**0 - 前缀**

* `Ctrl+b`

**1 - main**

* `new -s xx`
* `ls`
* `a -t xx`

**2 - session**

* `d`: 脱离会话

**3 - window**

* `c`: 创建新窗口
* `p`: 上一个
* `n`: 下一个
* `[0-9]`: 指定编号窗口

**4 - panel**

* `"`: 上线分割
* `%`: 左右分割
* `q`: 显示编号
* `z`: 最大化
