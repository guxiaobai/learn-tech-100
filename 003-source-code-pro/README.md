# Source Code Pro

### macOS

* `~/Library/Fonts`

### Ubuntu

```
[ -d /usr/share/fonts/opentype ] || sudo mkdir /usr/share/fonts/opentype
sudo git clone --depth 1 --branch release https://github.com/adobe-fonts/source-code-pro.git /usr/share/fonts/opentype/source-code-pro
sudo fc-cache -f -v
```

### Windows 10

* `c:\windows\fonts`
