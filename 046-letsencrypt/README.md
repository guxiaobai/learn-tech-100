## install

```bash
sudo snap install --classic certbot
sudo ln -s /snap/bin/certbot /usr/bin/certbot
```

## http

```
sudo python -m http.server 80
sudo certbot certonly --nginx -d "example.com" -m "sikuan.gu@gmail.com" --agree-tos
```
